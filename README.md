# Natural Gas Price Forecasting Dashboard

A portfolio-ready Python project with an interactive Streamlit dashboard for natural gas price analysis and forecasting.

## Features
- Interactive Streamlit dashboard
- Historical price analysis
- Trend + seasonality forecasting
- Date-based price estimator
- One-year future forecast
- Clean GitHub-ready structure

## Project Structure
```text
natural_gas_portfolio_project_pro/
├── app.py
├── model.py
├── gas_price_project.py
├── Nat_Gas.csv
├── requirements.txt
├── .gitignore
├── README.md
└── assets/
    └── screenshot_notes.txt
```

## Run the dashboard
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Run console version
```bash
python gas_price_project.py
```

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
